# path-undefined.github.io

Project setup:

```console
$ npm i
```

Start development content mock server:

```console
$ CONTENT_PATH=../path/to/blog-content npm run dev-env
```

Start development server:

```console
$ npm run dev
```

Build final artifact:

```console
$ npm run build
```

Release final artifact:

```console
$ npm run release
```

Release and publish the artifact to make it live:

```console
$ npm run release-and-publish
```
